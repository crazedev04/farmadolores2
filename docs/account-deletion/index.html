Solicitud de eliminacion de cuenta

Si queres eliminar tu cuenta y los datos personales asociados, segui los pasos de esta pagina.

Como solicitar la eliminacion

  1. Envia un correo a contacto@tu-dominio.com con el asunto Eliminar cuenta.
  2. Inclui estos datos para poder identificar tu cuenta:
    • Nombre y apellido (opcional).
    • Correo registrado en la app.
    • UID (si lo conoces).

Si preferis, podes usar este enlace de contacto: [agrega tu formulario aqui].

Que datos se eliminan

Se elimina la cuenta de autenticacion y los datos personales asociados (por ejemplo, nombre, correo y UID). El contenido publico puede mantenerse de forma anonima cuando sea necesario para el funcionamiento del servicio.

Plazos

Procesamos las solicitudes dentro de un plazo estimado de hasta 30 dias.
